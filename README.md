# 🤖 AI-BI Intelligent Ecosystem Dashboard

An extremely premium, responsive, and interactive **AI-BI (Artificial Intelligence - Business Intelligence) System Telemetry & Insights Dashboard**. 

This application was engineered specifically as an autonomous data orchestration display. It visualizes high-performance data pipeline workflows across multiple mock neural subagents, charts system computational metrics in real time, and provides deep business intelligence data breakdowns.

🔗 **Live Repository:** [github.com/Manikantapoola/ai-bi-ecosystem](https://github.com/Manikantapoola/ai-bi-ecosystem)

---

## 🌟 Key Features

* 📊 **Inference & Sync Telemetry**: Line chart modeling showing real-time query loads across autonomous server nodes and database pipelines.
* 🤖 **AI Subagent Pipeline Stream**: A step-by-step interactive workflow visualizing how datasets pass through:
  1. **Data Extraction Agent** (Crawls and structures database schemas)
  2. **Sentiment Engine Agent** (Cognitive language & sentiment parser)
  3. **Forecasting Agent** (Time-series neural trend forecasting)
  4. **BI Visualizer Node** (Generates rich reporting visualizations)
* ⚙️ **Ecosystem Controls**: Interactive sliders adjusting parameters like the Neural Temperature or Memory Buffer Size.
* 🛑 **Simulator Overrides**: Ability to trigger mock extreme loads (gpu warning events) and dynamically inject custom test queries down the pipeline.
* 🎨 **Premium Modern Design**: Designed with high-performance CSS grid grids, sleek custom scrollbars, glowing radial gradients, micro-animations, and a fully polished responsive dark mode layout.

---

## 🛠️ Technology Stack

* **Structure & Core**: HTML5 Semantic Markup
* **Styling & Layout**: Tailwind CSS (Utility-First CDN Engine)
* **Graphics & Charts**: Chart.js (Fluid Vector Visualization Canvas)
* **Icons & Vectors**: Lucide Icons CDN
* **Logic & Telemetry**: Native JavaScript (ES6+ Module Controls)

## 🚀 Easy Browser Execution (Recruiter Friendly)

This project has been built as a clean Single Page Application (SPA). This means it requires **zero installations, zero packages, and zero compilations** to run!

1. Clone or download the folder.
2. Double-click the **`index.html`** file in your local file explorer to open the dashboard directly in any modern browser!

## 🌐 Deploy to GitHub Pages (1-Minute Setup)

To make it accessible with a click to recruiters on your portfolio:
1. Go to your repository settings page on GitHub.
2. Select **Pages** from the left navigation menu.
3. Under the **Build and deployment** section, select `main` as the branch source and select `/` (root) as the folder.
4. Click **Save**. Within 60 seconds, GitHub will host it at `https://Manikantapoola.github.io/ai-bi-ecosystem`!
